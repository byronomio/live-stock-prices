Display live stock prices on your website with optional charts and a scrolling ticker.

== Description ==

Live Stock Prices allows you to display live stock prices on your WordPress website with optional charts and a scrolling ticker. It uses the IEX Cloud API to fetch real-time stock data.

Features:
* Display stock prices with optional charts
* Scrolling stock ticker
* Customizable update interval
* Easy to use shortcodes

Shortcodes:
1. `[live_stock_prices]` - Displays stock prices with optional charts
2. `[live_stock_prices_scrolling]` - Displays a scrolling stock ticker

To use this plugin, you need an API key from [IEX Cloud](https://iexcloud.io). [Sign up](https://iexcloud.io/cloud-login#/register) for an account and obtain your API key.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/live-stock-prices` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the 'Live Stock Prices' settings page under the 'Settings' menu to configure the plugin.
4. Enter your IEX Cloud API key, stock tickers, and other settings, then save.
5. Use the `[live_stock_prices]` or `[live_stock_prices_scrolling]` shortcodes to display stock prices on your website.
